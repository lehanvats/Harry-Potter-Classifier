There are 4 houses in harry potter.
Gryffindor: courageous, reckless, principled, confronts authority
Slytherin: ambitious, strategic, calculating, resourceful, networking
Hufflepuff: loyal, hardworking, fair, avoids conflict, team player
Ravenclaw: intellectual, curious, analytical, independent thinker

So using distilBERT, I created a sentiment based classifier, which takes a user input of them describing their personality, and the model categorizing them into one of the house archetypes.
First one just uses the categorization based on the key words given above using zero shot
Second one trains the model on 50 AI generated personality descriptions per house (200 total)

how to run:
open notebook on colab
import the personality_classification_dataset.txt dataset into the main colab folders

Use of AI has been done transparently and in redundant tasks.
They have been explicitly mentioned wherever used
The dataset is also fully generated by Claude as finding 200 different personality descriptions and categorizing would be too tedious otherwise
